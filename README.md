# PGCV Bioinformatics Workflow Dashboard

**Live URL:** [https://pgcv-bioinformatics-dashboard.vercel.app/](https://pgcv-bioinformatics-dashboard.vercel.app/)

*This project is currently under active development as part of the PGCV-BDML June–July 2026 Internship Program.*

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
